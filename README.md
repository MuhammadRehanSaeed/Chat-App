# ChatApp Frontend

A **real-time chat application frontend** built with **TypeScript**, supporting **global chat**, **one-to-one private messaging**, and **notifications**. Integrates with a **Java Spring Boot backend** using **WebSockets** and includes a **login system** for authenticated users.

---

## Features

- **User Authentication**: Secure login system for registered users.
- **Global Chat**: Send and receive messages visible to all online users.
- **Private Messaging**: One-to-one chat between users.
- **Real-time Notifications**: Get notified when new messages arrive.
- **WebSocket Integration**: Seamless real-time communication with the backend.
- **Responsive UI**: Works on desktops, tablets, and mobile devices.

---

## Tech Stack

- **Frontend**: TypeScript, React, HTML, CSS, JavaScript  
- **Backend**: Java Spring Boot with WebSocket support  
- **Real-time Communication**: WebSockets  
- **State Management**: React Context
- **Notifications**: Browser notifications for new messages  


## Getting Started

### Prerequisites

- Node.js v18+  
- npm or yarn  
- Backend server running (Spring Boot + WebSockets)
